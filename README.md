## Installation and Setup

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine. (`yarn` can be used too)  

Install dependencies:

`yarn`  

To Start Server:

`yarn start`  

Enter port:

`localhost:3000`
